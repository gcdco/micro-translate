Translate Microservice

This microservice translates text from a source language to English, and from English to a target language.

Translate to English

Uses google api to detect the source language. The POST request should have the source text sent in the body of the request as "translate". Make POST request to: "http://flip2.engr.oregonstate.edu:57666/translate/from".

Translate to Language

Takes an iso code for a target language to translate from English to target language. The POST request should have the source text sent in the body of the request as "translate" and the iso code of the target language as "iso". Make POST request to: "http://flip1.engr.oregonstate.edu:57666/translate/to".
